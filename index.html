CSS GRID LAYOUT TUTORIAL

"...offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning."

OBJECTIVE

The following tutorial provides an introductory overview of CSS Grid and is intended for students who have successfully completed a Level 1 Internet/Web Development class.

After going through the following tutorial, the student will be able to:

  • Set up a basic grid layout using CSS Grid.
  • Adjust the grid gap by modifying the grid-gap properties.
  • Adjust the placement of grid items.

WHAT CSS GRID IS

"CSS Grid Layout introduces a two-dimensional grid system to CSS. Grids can be used to lay out major page areas or small user interface elements."

From MDN Web Docs

WHAT CSS GRID ISN'T

Beginner web development students may have started learning about Bootstrap's grid system. This is not the same as CSS Grid. "Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases." With Boostrap, a developer will create a grid by using various divs. A div with a "container" class, then divs with "row" classes, and finally various divs inside each row with "col" or column classes. Items or content can then be placed in the various divs with the "col" classes.

From Boostrap

THE DIFFERENCE

CSS Grid provides an alternative to set up a grid, set up grid items, alter a grid, and is not restricted to 12 columns. In CSS Grid, the developer creates a div with a container class and then subsequent divs with "grid-item" classes. The developer is able to alter and modify the content within each div with the "grid-item" class. More in-depth CSS Grid examples are below.

ADJUST THE GRID GAP

The grid-gap property in CSS Grid refers to the space between columns and rows on the grid. Grid gaps cannot contain any content, but the size of the gaps can be adjusted. The grid-template-areas property changes the placement of grid items.
